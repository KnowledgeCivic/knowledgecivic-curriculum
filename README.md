# KnowledgeCivic Curriculum

Open, modular learning paths for people who want to understand public questions and act responsibly with others.

## Learning principles

- Start with a real question, not a vocabulary list.
- Teach learners how to inspect evidence, not what conclusion to memorize.
- Include reflection, dialogue, and a small public artifact.
- Make uncertainty and revision part of the lesson.
- Adapt examples to local language and context.
